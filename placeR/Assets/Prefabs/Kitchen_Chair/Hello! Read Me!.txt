The whole info about the model is bellow.

- Kitchen Chair model -

Model does not include any backgrounds or other objects used in preview images.

Model is Game-Ready.

It is a polygonal model (quads and tris), scaled and represented as the original object. Built in real world size (units - cm).

Model has 3 objects within, which are grouped. For easy use objects are named logically (using English language).

Model is set in world coordinates x0 y0 z0.

No special plugin needed to open scene.

All zip files contain textures (diffuse and bump maps .png 2048 x 2048). Model is UV mapped and all zip files contain not just textured UVs, 
but also raw UV maps in case You would like to make Your own custom textures for the chair.

Materials are Not included.

File Formats:
3Ds Max 2016 (contains standart materials)
.FBX  (contains standard materials)
.OBJ  (contains standard materials)
.3DS  (contains standard materials)

Warning: Depending on which software package you are using, the exchange formats (.fbx, .obj, .3ds) may not match the preview images exactly. 
Due to the nature of these formats, there may be some textures that have to be loaded by hand and possibly triangulated geometry.
